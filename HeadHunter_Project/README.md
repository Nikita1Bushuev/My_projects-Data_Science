# Анализ резюме соискателей с HeadHunter

## Описание проекта
Разведывательный анализ данных о резюме соискателей с платформы HeadHunter.

Проект  состоит из четырёх частей:

1. базовый анализ структуры данных

2. преобразование данных

3. разведывательный анализ

4. очистка данных
