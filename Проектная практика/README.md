# ♻️ Интеллектуальная система классификации отходов 

## 1. Проблема (Контекст и Актуальность)

**Неправильная сортировка отходов** является одной из ключевых проблем в области экологии и устойчивого развития. Несмотря на усилия по раздельному сбору мусора, значительная часть отходов по-прежнему попадает на свалки или сжигается без предварительной сортировки, что создает значительный вред экологии.

**Причины проблемы:**

- **Недостаточная осведомленность населения** - многие люди не знают, как правильно сортировать отходы;
- **Ошибки в сортировке** - люди могут ошибаться в определении типа материала;
- **Затраты на ручную сортировку** - предприятия по переработке тратят значительные ресурсы на ручное разделение отходов.

**Экологические последствия:**
- Загрязнение почвы и воды токсичными веществами (батарейки, электроника);
- Переполнение свалок неразлагаемыми материалами (пластик, стекло);
- Нерациональное использование ресурсов из-за низкого уровня переработки.

## 2. Решение проблемы

Разработка системы автоматической классификации типов отходов на основе компьютерного зрения использующей предобученные модели машинного обучения для идентификации материалов.

### Технические компоненты системы

| Элемент Системы | Входные Данные / Триггер | Функция / Реализация IT-Решения |
|----------------|--------------------------|--------------------------------|
| **Классификация отходов** | Фотографии отходов (с телефона, конвейера) | Модель машинного обучения **Augmented-Waste-Classifier-SigLIP2**, предобученная на 10 классах отходов. Задача: классификация изображений (image classification). |
| **Визуализация и Анализ** | Результаты классификации | Визуальное отображение типа отхода с указанием вероятности. Сбор статистики по типам обнаруженных материалов. |
| **Интеграция в процессы** | Данные классификации | Предоставление информации для: 1) мобильных приложений для граждан, 2) систем автоматической сортировки на заводах, 3) образовательных платформ. |

## 3. Использованная ML-модель

**Модель**: `prithivMLmods/Augmented-Waste-Classifier-SigLIP2`  
**Тип**: Vision-Language encoder (SigLIP2) для классификации изображений  
**Классы**: 10 типов отходов: Battery, Biological, Cardboard, Clothes, Glass, Metal, Paper, Plastic, Shoes, Trash  
**Источник**: Hugging Face Model Hub
